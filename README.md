# PDF Splitter
Simple Flask app. Display output from PDFMiner.

## Table of contents
* [General info](#general-info)
* [Technologies](#technologies)
* [Setup](#setup)

## General info
* Load PDF document
* Split it with PDFMiner
* Save to database
* Display splitted PDF content.

## Technologies
Project is created with:
* flask
* flask-sqlalchemy
* flask-admin
* pdfminer

## Setup
```
pip3 install -r requirements.txt
python app.py
```

Tests:
 ```
python test.py
```
